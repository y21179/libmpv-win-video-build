# libmpv-win-video-build

Self-built libmpv for Windows (zhongfly/mpv-winbuild, OpenSSL backend, native TLS 1.3).

Re-hosted binary for media_kit override package (`media_kit_libs_windows_video_tls13`).

## Source
- Built by: zhongfly/mpv-winbuild (based on shinchiro/mpv-winbuild-cmake)
- FFmpeg: `--enable-openssl` (native TLS 1.3 support)
- Architecture: x86_64-v3
